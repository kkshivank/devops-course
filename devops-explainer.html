So what is DevOps, really?

Imagine developers who write the code and operations teams who run the servers finally working as one team. DevOps is that culture plus a set of tools that lets you build, test, and ship software fast, safely, and automatically. This course walks you through the whole journey — writing code, tracking versions, automating tests, packaging apps, and deploying them to the cloud.

Follow the roadmap top to bottom — each block builds on the last.
